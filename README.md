# 🌟 Nirmal's Modern Blogger Site

Welcome to the repository for my personal, premium blogger site. This application is designed to be highly performant, visually stunning, and SEO-optimized, built with modern web development standards.

## 🚀 Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Radix UI (shadcn/ui)
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Deployment:** Netlify

## 🎯 Features

- **Blazing Fast Performance:** Server-rendered and statically generated utilizing App Router.
- **Premium Aesthetics:** Clean, modern UI using Tailwind CSS and interactive micro-animations through Framer Motion.
- **Dark Mode Support:** Built-in seamless theme toggling.
- **Markdown (MDX) Powered:** Allowing rich content integration in blog posts.
- **SEO Optimized:** Structured metadata implementation for prominent search engine visibility.

## 🏗️ Local Development

To get a local copy up and running, follow these simple building steps:

1. **Clone the repo**
   ```sh
   git clone https://github.com/iam-neo/YOUR_BLOG_REPO.git
   ```
2. **Install NPM packages**
   ```sh
   npm install
   ```
3. **Run the development server**
   ```sh
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
